# Waste Collection Route Optimizer

A full-stack app that optimizes waste collection routes and visualizes savings.

## Features
- Load pickup locations from CSV
- Optimize route with OR-Tools
- Visualize path on a map
- Show distance and fuel savings

## Quickstart
See Documentation/Quickstart.md.

## Tech Stack
- Frontend: React + Leaflet
- Backend: FastAPI (Python)
- Optimization: Google OR-Tools

## Data
Edit `Backend/data/pickups.csv` to add or change stops.
